Coleção Postman

Climbe API no Postman

Coleção que cobre toda a Climbe API — 9 pastas, 38 requests: autenticação, cargos, usuários, serviços, empresas, propostas, contratos e documentos.

⬇ Baixar a collection (.json)

Importar

  1. No Postman: Import → arraste o arquivo Squad-X-Climbe.postman_collection.json.
  2. Isso cria a collection Squad-X | Climbe já com as pastas e requests.

Usar

  1. Em Variables da collection, confira base_url (padrão http://localhost:8000) e ajuste admin_email / admin_password conforme o seu .env.
  2. Rode Auth → Login. O token é salvo em access_token automaticamente e os demais requests herdam o Bearer.
  3. Ajuste os id_* (ex.: id_empresa, id_proposta) para apontar os recursos nos endpoints com path param.

Fonte e gerador da coleção ficam em climbe-api/docs/postman/.
